HR Manager Pro

Sistema Completo de Gestão de Recursos Humanos
Desenvolvido com Django e Bootstrap

Conheça os Recursos

Gestão de Funcionários

Cadastro completo de funcionários com dados pessoais, profissionais e bancários.

Controle de Férias

Solicitação, aprovação e controle de férias com cálculo automático de dias.

Folha de Pagamento

Cálculo automático de salários, INSS, IRRF e demais descontos e proventos.

Avaliação de Desempenho

Avaliações periódicas com múltiplos critérios e notas automáticas.

Treinamentos

Gerenciamento de treinamentos e controle de participação dos funcionários.

Relatórios e Gráficos

Dashboard interativo com gráficos e exportação para Excel.

Começar a Usar

1. Acessar o Sistema

O HR Manager Pro está rodando em Django. Para acessar:

Desenvolvimento:
python manage.py runserver

Em seguida, acesse: http://127.0.0.1:8000

Credenciais de Teste:
  • Usuário: admin
  • Senha: admin123
2. Estrutura do Sistema

O sistema foi desenvolvido com:

  • Django 4.2.7 - Framework Python
  • Bootstrap 5 - Interface responsiva
  • SQLite - Banco de dados (desenvolvimento)
  • Chart.js - Gráficos interativos
  • Pandas - Exportação Excel
3. Funcionalidades Incluídas
  • Cadastro de Empresas, Departamentos e Cargos
  • Gestão completa de Funcionários
  • Controle de Férias e Faltas
  • Folha de Pagamento automática
  • Avaliações de Desempenho
  • Gestão de Documentos e Treinamentos
  • Benefícios e Advertências
  • Dashboard com gráficos
  • Exportação para Excel
